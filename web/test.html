Quick Start Guide

1. Make sure the server is running:

cd ~/skunkworks-production-agents/streamvoice/server
npm start

2. Access StreamVoice:

  • Open StreamVoice Main Interface
  • Open Debug Tool

3. What you should see:

  • ✅ Green "Connected" status at the top
  • ✅ Voice button is enabled (not grayed out)
  • ✅ WebSocket connection on port 8090
  • ✅ REST API on port 3030

4. Test a command:

  1. Click and hold the microphone button
  2. Say "Switch to gameplay"
  3. Release the button
  4. You should see the command recognized

5. If you're seeing "Disconnected":

  • Hard refresh: Ctrl+Shift+R
  • Clear browser cache
  • Try an incognito/private window
  • Check browser console (F12) for errors
  • Ensure you're using Chrome (required for voice recognition)
