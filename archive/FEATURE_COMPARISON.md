## New in StreamVoice Enhanced v0.3.0

### 🎚️ Audio Mixer
- Full volume control for all sources
- Visual sliders in web interface
- Voice commands: "increase mic volume", "desktop volume fifty percent"
- Quick mute/unmute buttons

### 🎬 Stream Deck Alternative
- One-click macros without hardware
- Pre-programmed sequences:
  - Stream starting routine
  - Stream ending routine
  - Raid mode activation
  - Subscriber celebrations
- Custom macro support

### 🎨 Advanced OBS Control
- **Transitions**: Change type and duration
- **Filters**: Toggle green screen, blur, color correction
- **Studio Mode**: Full preview/program control
- **Virtual Camera**: Start/stop for Zoom/Discord
- **Screenshots**: Voice-activated captures
- **Replay Buffer**: Save highlights on command

### 🎯 Enhanced Voice Recognition
- 70+ voice commands (vs 20 in v0.2)
- Better fuzzy matching
- Command suggestions
- Categorized command list

### 🖥️ Professional UI
- Tabbed interface
- Real-time OBS status
- Command categories
- Audio mixer panel
- Stream Deck grid

## Why StreamVoice Enhanced?

1. **Free Stream Deck Alternative**: Get hardware-level control without spending $149+
2. **Voice First**: Hands-free operation while gaming
3. **Web Based**: Works on any computer, no installation required
4. **Open Source**: Customize and extend as needed
5. **Kid Friendly**: Easy setup with EASY_TEST.bat

## Upgrade Path

Current v0.2 users can upgrade to Enhanced:
1. Download the latest release
2. Run EASY_TEST_ENHANCED.bat
3. Enjoy 50+ new commands and features!

The enhanced version is backwards compatible - all your favorite v0.2 commands still work!
