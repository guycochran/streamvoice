# StreamVoice Discord Server Setup Guide

## Server Name: StreamVoice Community

## Server Description
The official community for StreamVoice - a free Stream Deck alternative with voice control for OBS Studio. Get help, share feedback, and connect with other streamers!

## Channel Structure

### 📢 INFORMATION
- **#welcome** - Welcome message, server rules, and quick links
- **#announcements** - Official updates, new releases, important news
- **#roadmap** - Future features, development plans
- **#faq** - Frequently asked questions

### 💬 GENERAL
- **#general-chat** - General discussion about streaming and StreamVoice
- **#introductions** - New members introduce themselves
- **#showcase** - Share your streaming setup and clips
- **#off-topic** - Non-streaming related chat

### 🛠️ SUPPORT
- **#installation-help** - Help with downloading and installing
- **#bug-reports** - Report bugs and issues
- **#feature-requests** - Suggest new features
- **#troubleshooting** - Get help with problems

### 🎓 RESOURCES
- **#tutorials** - User-created guides and tips
- **#voice-commands** - Share custom commands
- **#stream-setups** - OBS configurations and tips
- **#macro-sharing** - Share your macro configurations

### 🔧 DEVELOPMENT
- **#beta-testing** - Early access and testing
- **#dev-updates** - Development progress updates
- **#contributors** - For those wanting to contribute code

### 🔊 VOICE CHANNELS
- **General Voice** - Casual chat
- **Support Voice** - Screen share for help
- **Streaming Together** - Co-streaming sessions

## Server Rules

### StreamVoice Community Rules

1. **Be Respectful** - Treat everyone with kindness and respect
2. **No Spam** - Avoid repeated messages, excessive mentions, or promotion
3. **Stay On Topic** - Use appropriate channels for discussions
4. **No Piracy** - Don't share pirated content or license keys
5. **Help Each Other** - We're here to support fellow streamers
6. **English Primary** - Main language is English for broader accessibility
7. **No NSFW Content** - Keep it family-friendly
8. **Report Issues** - Use GitHub for official bug reports

## Roles

### Staff Roles
- **@Developer** - Core development team (Red)
- **@Moderator** - Community moderators (Blue)
- **@Support Team** - Technical support volunteers (Green)

### Community Roles
- **@Beta Tester** - Early access to new features (Purple)
- **@Contributor** - Code/documentation contributors (Gold)
- **@Supporter** - Active helpful community members (Orange)
- **@Streaming Pro** - Experienced streamers who help others (Cyan)

### Auto-Assigned Roles
- **@Member** - All verified members
- **@Windows** - Windows users
- **@Mac** - Mac users (future)
- **@Linux** - Linux users (future)

## Bot Setup (Optional)

### Recommended Bots
1. **MEE6** - Welcome messages, auto-roles, moderation
2. **Ticket Tool** - Support ticket system
3. **GitHub Bot** - Sync GitHub issues/releases
4. **StreamCord** - Stream notifications

### Auto-Moderator Settings
- Block spam (5+ messages in 5 seconds)
- Block excessive mentions (5+)
- Block invite links (except GitHub)
- Word filter for inappropriate content

## Invite Link Settings
- No expiration
- Unlimited uses
- Grant @Member role on join
- Redirect to #welcome channel

## Server Boost Perks
- **Level 1**: Exclusive @Booster role, access to #booster-chat
- **Level 2**: Early access to beta features
- **Level 3**: Custom voice command requests, 1-on-1 support session

## Moderation Guidelines
1. Warnings for first offense
2. 24h mute for repeated issues
3. 7-day ban for serious violations
4. Permanent ban for extreme cases
