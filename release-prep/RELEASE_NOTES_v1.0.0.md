# StreamVoice v1.0.0 - First Official Release! 🎉

## 🚀 Introducing StreamVoice

StreamVoice is a **FREE alternative to Stream Deck** that lets you control OBS Studio with your voice. No $149 hardware required - just your voice and Chrome browser!

## ✨ Key Features

### 🎤 Voice Control (70+ Commands)
- Natural language processing - say "start recording" or "switch to gameplay"
- Hold-to-talk design for accurate recognition
- Works with any microphone
- Fuzzy matching for forgiving command recognition

### 🎛️ Stream Deck Alternative
- Visual macro grid with one-click actions
- Pre-programmed sequences for common tasks
- Emergency controls (panic button, quick mute)
- Customizable layout

### 🔌 Full OBS Integration
- Scene switching
- Source toggling
- Audio mixer control
- Recording/streaming control
- Studio mode support
- Filter management
- Virtual camera control

### 💻 Desktop App Features
- System tray integration
- Auto-start on Windows boot
- Runs in background
- Easy installer/uninstaller
- No configuration needed

## 📋 Requirements
- Windows 10 or 11
- OBS Studio 27.0 or newer
- Chrome browser (for voice control)
- Microphone

## 🛠️ Installation
1. Download `StreamVoice-Setup-1.0.0.exe`
2. Run the installer
3. Launch StreamVoice from Start Menu or Desktop
4. Enable OBS WebSocket (Tools → WebSocket Server Settings)
5. Open Chrome and navigate to the StreamVoice interface
6. Allow microphone access when prompted

## 🎯 Perfect For
- Streamers who want hands-free control

## 🐛 Known Limitations
- Chrome browser required (Web Speech API)
- OBS WebSocket must be enabled
- Scene names should match voice commands
- Windows only (Mac/Linux coming soon)
