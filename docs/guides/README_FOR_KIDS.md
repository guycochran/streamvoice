# 🎮 StreamVoice - For Young Streamers!

Control OBS with your voice - like Iron Man controls JARVIS! 🦾

## Super Easy Setup (5 minutes)

### What You Need:
- ✅ **OBS Studio** (free streaming software)
- ✅ **Chrome Browser** (for voice stuff)
- ✅ **Windows 10 or 11**
- ✅ **A microphone**

### Step 1: Download StreamVoice
1. Go to: https://github.com/guycochran/streamvoice
2. Click the green **Code** button
3. Click **Download ZIP**
4. Extract to your Desktop

### Step 2: Install Node.js (if needed)
1. Go to https://nodejs.org
2. Download the **LTS** version
3. Install it (just click Next, Next, Next)

### Step 3: Set Up OBS
1. Open **OBS Studio**
2. Go to **Tools → obs-websocket Settings**
3. Check ✅ **Enable WebSocket server**
4. Make sure password is **empty** (no password)
5. Click **OK**

### Step 4: Run StreamVoice
1. Open the StreamVoice folder on your Desktop
2. Double-click **EASY_TEST.bat**
3. Chrome will open automatically
4. You should see "Connected" in green!

## 🎤 How to Use It

1. **Hold down** the big microphone button
2. **Say a command** (speak clearly!)
3. **Let go** of the button
4. Watch OBS do what you said!

## 🗣️ Commands That Work

### Change Scenes
- **"Switch to gameplay"** - Goes to your game scene
- **"Switch to just chatting"** - Goes to chat scene
- **"Switch to starting"** - Starting soon screen
- **"Switch to break"** - BRB screen

### Recording
- **"Start recording"** - Starts recording your stream
- **"Stop recording"** - Stops the recording

### Audio
- **"Mute my mic"** - Turns off your microphone
- **"Unmute my mic"** - Turns on your microphone

## 🆘 If It Doesn't Work

### "Not Connected" or Red Status
1. Is OBS running? Start OBS first!
2. Did you enable WebSocket in OBS settings?
3. Try closing everything and run EASY_TEST.bat again

### Voice Commands Don't Work
1. Are you using Chrome? (Won't work in Edge/Firefox)
2. Did Chrome ask for microphone permission? Click "Allow"
3. Hold the button WHILE talking
4. Speak clearly and not too fast

### OBS Doesn't React
1. Check OBS has scenes named like "Gameplay" or "Just Chatting"
2. The scene names need to be similar to the commands
3. Try "Start recording" - that should always work

## 🎯 Pro Tips

1. **Practice the timing** - Hold, speak, release
2. **Name your OBS scenes** simple things like "Gameplay", "Chat", "BRB"
3. **Test with friends** - Take turns controlling the stream!
4. **Record yourself** using it - Share with other streamers!
