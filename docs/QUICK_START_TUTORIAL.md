# StreamVoice Quick Start Tutorial

## 🚀 From Download to First Command in 2 Minutes

### Prerequisites Check (30 seconds)
- [ ] Windows 10 or 11
- [ ] OBS Studio installed
- [ ] Chrome browser
- [ ] Microphone connected

### Step 1: Download & Extract (30 seconds)
1. Download: [StreamVoice-v0.2.0.tar.gz](https://github.com/guycochran/streamvoice/releases/download/v0.2.0/StreamVoice-v0.2.0.tar.gz)
2. Extract to your Desktop
3. You should see a `StreamVoice-v0.2.0` folder

### Step 2: OBS Setup (30 seconds)
1. Open OBS Studio
2. Go to: **Tools → obs-websocket Settings**
3. Check: ✅ **Enable WebSocket server**
4. Password: Leave empty
5. Port: Keep at 4455
6. Click **OK**

### Step 3: Install StreamVoice (30 seconds)
1. Open the StreamVoice folder
2. Double-click **INSTALL.bat**
3. Wait for "Installation complete!"
4. Press any key

### Step 4: Start StreamVoice (30 seconds)
1. Double-click **EASY_TEST.bat**
2. Chrome opens automatically
3. Look for green "Connected" status
4. If asked for microphone permission, click "Allow"

## 🎤 Your First Voice Command

1. **Hold down** the big microphone button
2. **Say clearly**: "Switch to gameplay"
3. **Release** the button
4. Watch OBS change scenes!

## 🎯 Quick Command Reference

Hold the button and say:

| Command | What it does |
|---------|--------------|
| "Switch to gameplay" | Changes to your game scene |
| "Be right back" | Shows BRB screen |
| "Start recording" | Starts OBS recording |
| "Mute my mic" | Mutes your microphone |
| "Emergency privacy" | Instantly hides everything |

## ❓ Troubleshooting

**"Not Connected" in red?**
- Is OBS running? Start it first!
- Did you enable WebSocket? (Step 2)

**Voice commands not working?**
- Using Chrome? (Required)
- Holding button WHILE speaking?
- Speaking clearly?

**Nothing happens in OBS?**
- Do you have a scene named "Gameplay"?
- Try "Start recording" - always works!
