### 2. Product Description

**Short Description** (60 chars):
"Voice control for OBS Studio. Say it, and it happens!"

**Full Description**:
```
StreamVoice brings voice control to OBS Studio, letting streamers control their production without interrupting gameplay or content.

🎤 How it works:
- Hold the microphone button
- Say "Switch to gameplay"
- OBS switches scenes instantly
- No hotkeys, no alt-tabbing!

✨ Key Features:
• 20+ voice commands out of the box
• 2-minute setup with EASY_TEST.bat
• Works with any microphone
• Chrome-based (uses Web Speech API)
• Hold-to-talk for accuracy
• Real-time WebSocket connection to OBS

🎮 Perfect for:
• Gamers who can't pause mid-match
• Content creators juggling multiple apps
• Streamers with mobility considerations
• Anyone tired of remembering hotkeys

📦 Current Status:
Beta v0.2.0 - Looking for feedback from the streaming community!

The project is open source and will always be free. Built by a dad whose 13-year-old wanted to stream like a pro!
```
