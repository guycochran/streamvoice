# How to Create the GitHub Release

## Quick Steps:

1. Go to: https://github.com/guycochran/streamvoice/releases/new

2. Fill in these fields:
   - **Tag version**: `v0.2.0`
   - **Target**: `main` (should be default)
   - **Release title**: `StreamVoice v0.2.0 - First Beta Release! 🎉`

3. In the description box, copy and paste the contents of `RELEASE_NOTES_v0.2.0.md`

4. Under "Attach binaries", drag and drop: `StreamVoice-v0.2.0.tar.gz`

5. Check: ✅ **This is a pre-release** (since it's beta)

6. Click: **Publish release**

## That's it!

The release will be live at:
https://github.com/guycochran/streamvoice/releases/tag/v0.2.0

Beta testers can then download directly from:
https://github.com/guycochran/streamvoice/releases/download/v0.2.0/StreamVoice-v0.2.0.tar.gz
