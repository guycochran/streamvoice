# StreamVoice Beta Tester Packet

## Beta Invite

I'm inviting a small group to test `StreamVoice`, a desktop voice control app for OBS Studio.

It lets you control OBS with short voice commands, hotkeys, and quick actions. This is a closed beta focused on:
- real-world usability
- voice accuracy
- operator trust
- setup clarity

## Who This Beta Is For

- OBS Studio users on Windows
- people with a microphone they can select in-app
- streamers, presenters, podcasters, churches, or live-event operators
- people willing to spend 15 to 20 minutes testing and give direct feedback

## What You Need

- Windows PC
- OBS Studio with OBS WebSocket enabled
- microphone connected
- 15 to 20 minutes
- willingness to test in a real room, not just a perfect quiet environment

## What To Install

- the latest `StreamVoice` beta build
- OBS Studio running locally

## Setup Instructions

1. Open OBS Studio.
2. Make sure OBS WebSocket is enabled.
3. Launch StreamVoice.
4. Go to `Settings`.
5. Select your microphone.
6. Confirm OBS host and port are correct.
7. Click `Test Connection`.
8. Leave `Basic Live Control` selected unless you specifically want to test `Studio Workflow`.
9. If desired, set a global voice hotkey.
10. Return to the main screen and test voice commands.

## What To Test

Please run this checklist.

### 1. Basic Connection

- Does StreamVoice connect to OBS?
- Does the selected microphone show input activity?

### 2. Core Voice Commands

- `mute mic`
- `unmute mic`
- `start stream`
- `stop stream`
- `start recording`
- `stop recording`

### 3. Scene Switching

- `switch to break`
- `switch to camera 1`
- `switch to camera 2`
- `go to powerpoint` or another mapped scene you actually use

### 4. Audio Adjustments

- `turn up the mic`
- `turn down the mic`
- `turn up the audio`
- `turn down the audio`

### 5. Quick Actions

- `Start Session`
- `End Session`
- `Emergency Mute`
- `Screenshot`

### 6. Hotkey

- assign the hotkey
- trigger voice capture with it
- confirm it feels usable

### 7. Real-World Test

- try in a normal room with some background noise
- note any commands that were misunderstood or felt too slow

## Known Beta Notes

- This is a closed beta, not a final release.
- `Basic Live Control` is the default and recommended mode.
- `Studio Workflow` is for more advanced preview/program style operation.
- Voice works locally on-device.
- Accuracy may vary with room noise, mic quality, and phrasing.
- The most important feedback is trust:
  - did it do the right thing
  - did it fail clearly
  - would you risk using it live

## Recommended Beta Mix

The best small beta group is:

- 2 streamers or gamers
- 1 presenter or educator
- 1 podcast or multi-camera user
- 1 church or live-event operator if possible

This beta is not about collecting generic opinions.
