# Git Operations I Can Handle

Now that SSH is configured, I can handle all these git operations for you:

## Basic Operations:
- `git add .` - Stage changes
- `git commit -m "message"` - Commit with descriptive messages
- `git push origin main` - Push to GitHub
- `git pull` - Get latest changes
- `git status` - Check repository status

## Branch Operations:
- `git checkout -b feature-branch` - Create new branches
- `git merge` - Merge branches
- `git branch -d branch-name` - Delete branches

## Advanced Operations:
- `git tag v1.0.0` - Create version tags
- `git push --tags` - Push tags to GitHub
- `git log --oneline` - View commit history
- `git diff` - See what changed

## Collaboration:
- `git fetch` - Get remote changes without merging
- `git remote -v` - Check remote repositories
- `git clone` - Clone any repository
