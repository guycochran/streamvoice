# StreamVoice v0.3.0 - Stream Deck Alternative! 🎯

Transform OBS control with 70+ voice commands and Stream Deck-like features - all for FREE!

## 🚀 What's New in v0.3.0

This is a MASSIVE update that transforms StreamVoice into a complete Stream Deck alternative:

### 🎚️ Stream Deck Alternative Mode
- **One-Click Macros** - Just like a $149 Stream Deck, but free!
- **Pre-programmed sequences**:
  - Stream Starting Setup (mute mic → starting scene → countdown → go live)
  - Stream Ending Setup (ending scene → stop recording → stop stream)
  - Raid Mode (special scene + chat focus)
  - Subscriber Celebrations (trigger alerts)
- **Emergency Buttons** - Instant privacy mode, emergency mute

### 🎛️ Professional Audio Mixer
- Visual volume sliders for all audio sources
- Real-time volume control with voice: "Increase mic volume"
- Quick mute/unmute buttons
- Set specific levels: "Mic volume fifty percent"
- Control desktop audio separately

### 🎬 Advanced OBS Controls
- **Transitions**: Change type (Cut, Fade, Slide, Stinger) and duration
- **Filters**: Toggle green screen, blur, color correction with voice
- **Studio Mode**: Full preview/program control
- **Virtual Camera**: Start/stop for Zoom, Discord, Teams
- **Screenshots**: Voice-activated captures
- **Replay Buffer**: Save highlights on command

### 🎯 Enhanced Voice Recognition
- **70+ Commands** (up from 20 in v0.2)
- Better fuzzy matching algorithm
- Command suggestions when unclear
- Organized command categories
- Improved error handling

### 🖥️ Professional UI Overhaul
- **Tabbed Interface**:
  - Voice Control tab
  - Stream Deck tab (macro grid)
  - All Commands tab (categorized)
  - Audio Mixer tab
  - OBS Status tab
- Real-time OBS connection status
- Visual feedback for all actions
- Mobile-responsive design

## 📦 Installation

### Upgrade from v0.2.0
1. Download `StreamVoice-v0.3.0.tar.gz`
2. Extract over existing installation
3. Run `EASY_TEST_ENHANCED.bat` for new features

### Fresh Install
1. Download `StreamVoice-v0.3.0.tar.gz`
2. Extract to Desktop
3. Run `INSTALL.bat` (one time)
4. Choose your launcher:
   - `EASY_TEST.bat` - Basic version (20 commands)
   - `EASY_TEST_ENHANCED.bat` - Full version (70+ commands)
