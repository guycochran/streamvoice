# StreamVoice Demo Video Script (60-90 seconds)

## Video Title: "Control OBS With Your Voice in 60 Seconds!"

### Scene 1: Hook (0-5 seconds)
**Visual**: Split screen - left shows frantic mouse clicking/alt-tabbing, right shows calm streamer
**Script**: "Tired of this? *shows alt-tabbing* What if you could do this instead?"

### Scene 2: Introduction (5-15 seconds)
**Visual**: StreamVoice interface appears
**Script**: "Meet StreamVoice - voice control for OBS. Watch how easy it is..."

### Scene 3: Live Demo (15-45 seconds)
**Visual**: Full screen OBS + small webcam overlay showing streamer

**Action sequence:**
1. **Starting** (hold mic button)
   - Say: "Switch to starting soon"
   - OBS switches to Starting scene

2. **Going live** (hold mic button)
   - Say: "Switch to gameplay"
   - OBS switches to Gameplay scene

3. **Recording** (hold mic button)
   - Say: "Start recording"
   - OBS recording indicator turns on

4. **Quick break** (hold mic button)
   - Say: "Be right back"
   - OBS switches to BRB scene

5. **Emergency** (hold mic button)
   - Say: "Emergency privacy"
   - OBS instantly switches to safe scene

### Scene 4: Setup (45-55 seconds)
**Visual**: Quick montage of setup steps
**Script**: "Setup takes 2 minutes. Download, extract, run INSTALL.bat, then EASY_TEST.bat. Done!"

### Scene 5: Call to Action (55-60 seconds)
**Visual**: Download button and beta tester text
**Script**: "Free beta available now. Be one of the first 10 testers!"

## Production Notes

### Required OBS Scenes:
- Starting Soon (animated countdown or "Starting Soon" text)
- Gameplay (game capture)
- Just Chatting (webcam fullscreen)
- BRB (Be Right Back screen)
- Ending (Thanks for watching screen)

### Recording Tips:
- Use OBS to record the demo (meta!)
- Show the mic button being held clearly
- Add captions for voice commands
- Keep energy high and pacing quick
- Background music: upbeat, non-copyrighted

### Post-Production:
- Add animated arrows pointing to important UI elements
- Highlight successful commands with green checkmarks
- Speed up the download/extract portion to 2-3 seconds
- End with GitHub URL as text overlay
