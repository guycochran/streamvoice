# Enable GitHub Pages for StreamVoice

To make the landing page live at https://guycochran.github.io/streamvoice/

1. Go to: https://github.com/guycochran/streamvoice/settings/pages
2. Under "Source", select: **Deploy from a branch**
3. Branch: **main**
4. Folder: **/docs**
5. Click **Save**

The site will be live in 2-10 minutes at:
https://guycochran.github.io/streamvoice/

This gives you a professional landing page for sharing!
